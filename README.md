# রক্তবন্ধন (RoktoBondhon) — Phase 1 সেটআপ গাইড

## ⚠️ গুরুত্বপূর্ণ — আগের ডাটাবেস থাকলে
এই ভার্সনের `database.sql`-এ আগের ছোট টেবিল **DROP করে** নতুন বড় স্ট্রাকচার বানানো হয়। আগে যদি টেস্ট অ্যাকাউন্ট বানিয়ে থাকো, সেগুলো এই ইম্পোর্টের পর হারিয়ে যাবে — চিন্তার কিছু নেই যদি ওগুলো শুধু টেস্ট ডেটা হয়।

## ১। ডাটাবেস আপডেট করো
phpMyAdmin-এ গিয়ে SQL ট্যাবে নতুন `database.sql`-এর পুরো কনটেন্ট পেস্ট করে Go চাপো। এতে ৫টা টেবিল তৈরি হবে: `donors`, `emergency_requests`, `hospitals`, `ambulances`, `admins`।

## ২। কোড পুশ করো (Termux)
```bash
cd ~
unzip -o blood-donor-finder.zip
cd blood-donor-finder
git add .
git commit -m "Phase 1: full feature set"
git push origin main
```
(GitHub Secrets আগে থেকেই বসানো আছে ধরে নিচ্ছি — না থাকলে আগের গাইড অনুযায়ী বসিয়ে নাও)

## ৩। ডিপ্লয় শেষ হলে — অ্যাডমিন অ্যাকাউন্ট বানাও
ব্রাউজারে যাও: `https://roktobondhon.rf.gd/admin/setup.php`
- একটা ইউজারনেম ও পাসওয়ার্ড (কমপক্ষে ৮ অক্ষর) দিয়ে অ্যাডমিন অ্যাকাউন্ট বানাও।
- **এরপর সাথে সাথেই** File Manager বা FTP দিয়ে `admin/setup.php` ফাইলটা সার্ভার থেকে ডিলিট করে দাও — নাহলে যে কেউ এই লিংকে গিয়ে পেজটা দেখতে পারবে (যদিও দ্বিতীয়বার অ্যাকাউন্ট বানাতে পারবে না, তবু না রাখাই নিরাপদ)।
- তারপর থেকে `https://roktobondhon.rf.gd/admin/login.php` দিয়ে লগইন করবে।

## নতুন কী কী যুক্ত হলো (Phase 1)
- বিভাগ → জেলা → উপজেলা cascading dropdown (পুরো ৪৯৫ উপজেলা সহ)
- প্রোফাইল ছবি আপলোড (JPG/PNG, ২MB পর্যন্ত)
- হোমপেজে লাইভ স্ট্যাটিস্টিকস, ব্লাড গ্রুপ ব্রেকডাউন, সাম্প্রতিক ডোনার, Compatibility Chart
- জরুরি রক্ত রিকোয়েস্ট (পাবলিক পোস্ট, অ্যাডমিন অনুমোদনের পর সাইটে দেখা যায়)
- রক্তদান কুলডাউন (১২০ দিন) — Progress Bar সহ, "আজ রক্ত দিয়েছি" বাটন দিয়ে আপডেট হয়
- ব্লাড কার্ড — QR কোড সহ, যেকোনো জায়গায় স্ক্যান করে ভেরিফাই করা যায় (`/verify.php`)
- হাসপাতাল ও অ্যাম্বুলেন্স ডিরেক্টরি (অ্যাডমিন থেকে যোগ করা যায়)
- নারী ডোনারের ফোন নম্বর সার্চ রেজাল্টে লুকানো থাকে
- অ্যাডমিন প্যানেল: ড্যাশবোর্ড, ডোনার ম্যানেজমেন্ট (ভেরিফাই/ব্লক/ডিলিট), রিকোয়েস্ট অনুমোদন, হাসপাতাল/অ্যাম্বুলেন্স ম্যানেজমেন্ট
- CSRF প্রোটেকশন সব ফর্মে

## এখনো বাকি (টাকা বা বড় সেটআপ লাগবে বলে Phase 1-এ নেই)
- SMS/WhatsApp/Telegram অ্যালার্ট (পেইড গেটওয়ে লাগবে)
- Google/Facebook লগইন (Google/Meta Developer অ্যাকাউন্ট সেটআপ লাগবে)
- GPS ভিত্তিক "কাছের ডোনার" আর Google Maps (Google বিলিং অ্যাকাউন্ট লাগবে)
- AI ফিচার (পেইড AI API লাগবে)
- নেটিভ Android/iOS অ্যাপ (চাইলে এটা পরে আলাদাভাবে বানানো যাবে)
- থ্যালাসেমিয়া/প্লাটিলেট/প্লাজমা আলাদা সিস্টেম, ব্লগ, রিভিউ, র‍্যাংকিং — এগুলো চাইলে ধাপে ধাপে পরে যোগ করা যাবে

## নিরাপত্তা মনে রাখার বিষয়
- `config.php` কখনো git-এ কমিট হয় না, পাসওয়ার্ড সবসময় GitHub Secrets-এ থাকে।
- `admin/setup.php` ব্যবহারের পর অবশ্যই ডিলিট করে দিও।
- টোকেন/পাসওয়ার্ড কখনো চ্যাটে শেয়ার করবে না।
